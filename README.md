# Project MEDICA

Project MEDICA is a medical imaging research project focused on lung nodule detection and classification in chest X-rays using deep learning models.

The project aims to develop, evaluate, and demonstrate a research prototype that compares a CNN baseline model with a lightweight Vision Transformer model.

## Project Objective

To investigate whether a lightweight Vision Transformer can match or outperform a heavier CNN baseline for lung nodule detection while offering better interpretability and practical deployment potential.

## Team Roles

### Data & Model Architect

Responsible for data preprocessing, model development, training, and optimization.

### Evaluation & System Engineer

Responsible for model evaluation, statistical analysis, prototype development, system workflow, risk analysis, and documentation.

## Current Phase

Phase 0: Scope and Foundation

The current focus is to establish the development environment, understand the dataset, define the evaluation framework, and prepare the project structure.

## Repository Structure

- `docs/`: project documentation, experiment logs, risk register, evaluation notes
- `src/data/`: data loading and preprocessing scripts
- `src/evaluation/`: model evaluation scripts and metrics
- `src/models/`: model architecture and training code
- `src/app/`: prototype application code
- `notebooks/`: exploratory analysis and experiments
- `configs/`: experiment and training configurations
- `results/`: generated outputs, figures, and evaluation tables

## Important Note

This project is for research and educational purposes only.  
It is not intended for clinical diagnosis or medical decision-making.
